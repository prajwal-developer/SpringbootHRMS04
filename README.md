# HRMS Management System

A Human Resource Management System (HRMS) built using Spring Boot.

## Features

- Employee Management
- Attendance Management
- Leave Management
- Payroll Management
- AI Assistant Integration

## Tech Stack

- Java 17
- Spring Boot
- Spring Data JPA
- Hibernate
- MySQL
- Maven
- HTML/CSS/JavaScript

## API Endpoints

### Employee
- POST /employees
- GET /employees
- GET /employees/{id}
- PUT /employees/{id}
- DELETE /employees/{id}

### Attendance
- POST /attendance
- GET /attendance
- GET /attendance/{id}

### Leave
- POST /leave
- GET /leave
- PUT /leave/{id}

### Payroll
- POST /payroll
- GET /payroll
